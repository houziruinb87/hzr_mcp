# Open-XiaoAI 备份说明（连接远端小智可用）

本目录下当前可用的配置已做备份，**说明连接远端小智官方服务（tenclass.net）没问题**。

## 备份文件

| 文件 | 说明 |
|------|------|
| `config.backup-official-xiaozhi-ok.py` | 当前 config.py 的备份（官方小智 OTA/WebSocket 地址 + vad boost） |
| `docker-compose.backup-official-xiaozhi-ok.yml` | 当前 docker-compose 的备份（最简部署、无自建小智网络） |

## 恢复方式

若之后改乱了想还原到这份可用配置：

```bash
# 在 oxa-server 目录或 NAS 的 Open-XiaoAI-Server 目录下
cp config.backup-official-xiaozhi-ok.py config.py
cp docker-compose.backup-official-xiaozhi-ok.yml docker-compose.yml
docker compose up -d
# 或仅更新 config 时：上传 config.py 后 docker restart open-xiaoai-xiaozhi
```

## 配置要点（当前可用）

- **小智**：使用官方 `https://api.tenclass.net` / `wss://api.tenclass.net`，未接自建 xiaozhi-esp32-server。
- **唤醒词**：豆包豆包、你好小智、小智同学、hi siri。
- **VAD**：`boost: 150`（小爱录音放大）、`threshold: 0.05`。
