## cuco.plug.v3（MIOT 插座/灯）

**本型号设备**：乌龟灯、走廊灯。

走 MIOT 协议，用 `miiocli miotdevice`，开关属性为 **siid=2, piid=1**。参数按 Python 字面量传，布尔用 **True/False（大写）**。

### 乌龟灯（已验证）

- IP: `192.168.1.101`  
- token: `YOUR_MIOT_DEVICE_TOKEN`

**开灯（已验证）：**

```bash
miiocli miotdevice --ip 192.168.1.101 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":True}]'
```

**关灯（已验证）：**

```bash
miiocli miotdevice --ip 192.168.1.101 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":False}]'
```

### 走廊灯（同型号，同一套指令，仅 IP/token 不同）

- IP: `192.168.1.104`  
- token: `YOUR_MIOT_DEVICE_TOKEN`

**开灯：**

```bash
miiocli miotdevice --ip 192.168.1.104 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":True}]'
```

**关灯：**

```bash
miiocli miotdevice --ip 192.168.1.104 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":False}]'
```

### 定时/延时（cuco.plug.v3）

- 官方 API 与发现步骤见 [延时与定时API说明.md](延时与定时API说明.md)。
- **已明确**：**piid=3** = 延时时长（写 5 时米家 App 显示 5 分钟）；**piid=1** = 延时开启开关（False=关，True=开）。仅设 piid=3 只会改 App 里的「延迟时间」，不会触发延时，需**同时把 piid=1 设为 True** 才会执行「延时开启」。

**验证情况**：  
- 设 piid=1=False、piid=3=2 均返回 code 0，但 App 的「延时开启」「延迟时间」均无变化；  
- **全网搜索 + 抓包验证**后结论：  
  - **功能实现方式**：米家 App 的「延时开启/关闭」**不是直接调用设备的 miot 属性**，而是通过**米家云端 API 创建定时场景**（`/app/scene/edit`）实现的。场景由云端计时，到时下发开/关命令给设备。  
  - **本地 API 调用无效原因**：设备的 siid=4 属性（延时相关）无法通过本地 `set_properties` 触发倒计时逻辑，只能由云端场景驱动。  
  - **替代方案**：用外部脚本 `time.sleep()` + 发开/关命令实现延时（简单、本地、不依赖云端）。  
- **当前已验证可用**：仅**开关（siid=2, piid=1）**。详见 [延时与定时API说明.md](延时与定时API说明.md) 第八节"抓包验证结论"。

#### 本地延时实现（hzr_mcp 集成）

已在 `hzr_mcp.py` 中实现外部脚本延时方案，支持语音控制：

**乌龟灯延时功能**：
- `wuguideng_delayed_on(delay, unit)` - 延时开启（支持秒/分钟/小时）
- `wuguideng_delayed_off(delay, unit)` - 延时关闭（支持秒/分钟/小时）
- `wuguideng_cancel_delay()` - 取消延时任务

**语音示例**：
- 「30秒后开启乌龟灯」→ 30秒后自动开启
- 「5分钟后关闭乌龟灯」→ 5分钟后自动关闭
- 「1小时后关乌龟灯」→ 1小时后自动关闭
- 「取消乌龟灯延时」→ 取消所有延时任务

**特性**：
- ✅ 新指令自动覆盖旧的延时任务
- ✅ 本地计时，不依赖云端和网络
- ✅ 支持多设备独立延时（乌龟灯、走廊灯互不影响）

## yodq.humidifier.yo001

**本型号设备**：YO Humidifier（YO 加湿器）。

### 设备信息
- IP: `192.168.1.103`  
- Token: `YOUR_MIOT_DEVICE_TOKEN`
- 硬件: ESP32C3
- 固件: 1.0.6

### 已验证的控制命令

**开关控制**：

```bash
# 打开加湿器
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":True}]'

# 关闭加湿器
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":1,"value":False}]'

# 查询开关状态
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command get_properties '[{"siid":2,"piid":1}]'
```

### 已验证的属性

| siid | piid | 功能 | 类型 | 可用值 | 状态 |
|------|------|------|------|--------|------|
| 2 | 1 | 电源开关 | 布尔 | True/False | ✅ 已验证 |
| 2 | 9 | 场景模式 | 整数 | 1=睡眠, 2=自动, 3=强劲 | ✅ 已验证 |
| 2 | 10 | 手动档位 | 整数 | 1=弱档, 2=中档, 3=强档 | ✅ 已验证 |
| 4 | 1 | 功能锁（童锁） | 布尔 | True=锁定, False=解锁 | ✅ 已验证 |

**控制模式说明**：
- **手动档位**（piid=10）：固定风量大小，适合手动控制
  - 1 = 弱档（小风量）
  - 2 = 中档（中风量）
  - 3 = 强档（大风量）

- **场景模式**（piid=9）：智能调节风量，适合自动控制
  - 1 = 睡眠模式（静音运行）
  - 2 = 自动模式（根据湿度自动调节）
  - 3 = 强劲模式（最大加湿）

**注意**：两种模式可以分别设置，但通常只使用其中一种。

**手动档位控制命令**：

```bash
# 设置弱档
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":10,"value":1}]'

# 设置中档
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":10,"value":2}]'

# 设置强档
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":10,"value":3}]'
```

**场景模式控制命令**：

```bash
# 设置睡眠模式
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":9,"value":1}]'

# 设置自动模式
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":9,"value":2}]'

# 设置强劲模式
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":2,"piid":9,"value":3}]'
```

**童锁控制命令**：

```bash
# 开启童锁
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":4,"piid":1,"value":True}]'

# 关闭童锁
miiocli miotdevice --ip 192.168.1.103 --token YOUR_MIOT_DEVICE_TOKEN raw_command set_properties '[{"siid":4,"piid":1,"value":False}]'
```

**未支持的属性**：
- siid=2, piid=2：只读，无法通过局域网设置
- siid=2, piid=9：只读（始终为 1）
- siid=3：整个服务不支持（code=-4003，无传感器数据）
- siid=5, piid=1：只读，无法通过局域网设置

**注意**：
- python-miio 库尚未官方支持此型号，需使用 `raw_command`
