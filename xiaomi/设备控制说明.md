## cuco.plug.v3（MIOT 插座/灯）

**本型号设备**：乌龟灯、走廊灯。

走 MIOT 协议，用 `miiocli miotdevice`，开关属性为 **siid=2, piid=1**。参数按 Python 字面量传，布尔用 **True/False（大写）**。

### 乌龟灯（已验证）

- IP: `192.168.50.220`  
- token: `e6c9cc7e3041298eb0bd32ad75f11d33`

**开灯（已验证）：**

```bash
miiocli miotdevice --ip 192.168.50.220 --token e6c9cc7e3041298eb0bd32ad75f11d33 raw_command set_properties '[{"siid":2,"piid":1,"value":True}]'
```

**关灯（已验证）：**

```bash
miiocli miotdevice --ip 192.168.50.220 --token e6c9cc7e3041298eb0bd32ad75f11d33 raw_command set_properties '[{"siid":2,"piid":1,"value":False}]'
```

### 走廊灯（同型号，同一套指令，仅 IP/token 不同）

- IP: `192.168.50.181`  
- token: `5d3398d592761f5459ba83b549409901`

**开灯：**

```bash
miiocli miotdevice --ip 192.168.50.181 --token 5d3398d592761f5459ba83b549409901 raw_command set_properties '[{"siid":2,"piid":1,"value":True}]'
```

**关灯：**

```bash
miiocli miotdevice --ip 192.168.50.181 --token 5d3398d592761f5459ba83b549409901 raw_command set_properties '[{"siid":2,"piid":1,"value":False}]'
```

### 定时/延时（cuco.plug.v3）

该型号通常支持定时开、定时关、延时开、延时关，对应 **siid=4** 的定时/倒计时属性（具体 siid/piid 需本机确认）。先查询是否支持：

```bash
miiocli miotdevice --ip 192.168.50.220 --token e6c9cc7e3041298eb0bd32ad75f11d33 raw_command get_properties '[{"siid":4,"piid":1},{"siid":4,"piid":2},{"siid":4,"piid":3},{"siid":4,"piid":5}]'
```

常见约定：piid 3 = countdown（秒）、1 = on_duration、2 = off_duration、4 = task_switch、5 = countdown_info（只读）。控制时用 `set_properties` 写对应 siid/piid/value。
